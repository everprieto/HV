# 📄 Resume Analyzer API (FastAPI + OpenAI)

## 🇺🇸 English Version

### 🚀 Overview

This is a public API built with FastAPI that:

- Accepts a PDF resume file
- Extracts the text
- Sends it to OpenAI
- Returns structured JSON data

The API uses the OpenAI model `gpt-4.1` to extract structured resume information.

---

### 📦 Tech Stack

- Python
- FastAPI
- OpenAI API
- PyPDF
- Uvicorn
